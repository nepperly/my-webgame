何时落下的雪花

一、引言

因为一些原因在网络上交友几乎是你唯一的交友方式。

有一天你意外点进了一个修仙论坛，一开始你点进去的时候根本看不懂他们在说什么东西，虽然现在也是，但就在你要退出这个论坛的时候你刷新出来了一个新帖子。

【刚刚化成人形！现在的人类社会好可怕！】

你被引起了兴趣，点进去看了看。楼主是一只小狐狸，最近刚刚化成人形，就像她所说的那样，人类的社会发生了很大的变化。下面很快就有人回了楼主，但很多都是什么招徒弟，一键帮你成为狐仙……你看一眼都能知道是修仙界的诈骗。你不知道自己能帮到对方什么，就留下了一句加油和自己能帮到对方的话关闭了界面。

缘分就是这样的不可思议。你们就这样聊了起来。

小狐狸很可爱也愿意告诉你他们修仙论坛相关的东西。这个修仙论坛是由各个妖怪和修仙的人类创建起来的平台，妖怪可以不用现代社会的这些电子设备在论坛上互动，这也是为什么小狐狸只能用论坛和你聊天的原因。

关于修仙的那些你不是很关心，不过你很喜欢这只可爱的小狐狸。

不过两天前她突然开始不回你的消息。一开始你以为是她的作息改变或者没有看到消息，但一天过去，两天过去，你意识到不对，开始担心起来，却发现自己几乎没有对方的线索，找人求助也没有意义。你觉得自己不能坐以待毙，开始对这个论坛开始了你的调查想要找到小狐狸的线索……

二、玩法

1、游戏目的

通过浏览我是小神仙论坛

  • 找到小狐狸的线索
  • 找回小狐狸
  • 或许有一点彩蛋？

2、具体玩法

阅读论坛页面，找到可能的线索进行进一步的搜索，去到下一个界面收集线索，重复以上的步骤直到达到游戏目的。

3、温馨提示

  1. 可以新建一个文档来记录所看到的线索
  2. 不懂的地方多多使用搜索引擎哦，即使大部分架空，但是部分的小解密可能会用到
  3. 不一定只有没听过的专有名词才是线索，请尽可能地搜索和尝试
  4. 游戏中没有跳脸或血腥画面（也并没有技术力制作这些），游戏不包含任何音效
  5. 由于本人第一次制作相关游戏内容，因此游戏内的搜索较为单一，大部分以人名为主
  6. 为了您的正常游玩，请不要打开页面的源代码或是直接修改网页的链接
  7. 那么最后，请打开你的电脑开始游戏吧
